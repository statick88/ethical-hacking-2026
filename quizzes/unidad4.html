Preguntas de Opción Múltiple

Pregunta 1 (Básica)

¿Qué protocolo usa Kerberos por defecto en Active Directory?

  1. LDAP
  2. SMB
  3. Kerberos
  4. NTLM

Respuesta correcta: c

Pregunta 2 (Básica)

¿Qué es Kerberoasting?

  1. Atacar el KDC
  2. Crackear tickets TGS offline
  3. Robar credenciales
  4. Enumerar usuarios

Respuesta correcta: b

Pregunta 3 (Intermedia)

¿Cuál es el puerto default de LDAP?

  1. 389
  2. 445
  3. 636
  4. 88

Respuesta correcta: a

Pregunta 4 (Intermedia)

¿Qué herramienta se usa para visualizar relaciones en AD?

  1. Mimikatz
  2. BloodHound
  3. CrackMapExec
  4. Rubeus

Respuesta correcta: b

Pregunta 5 (Avanzada)

¿Qué técnica permite moverse lateralmente usando hashes NTLM?

  1. Kerberoasting
  2. Pass-the-Hash
  3. Golden Ticket
  4. DCSync

Respuesta correcta: b

Pregunta 6 (Básica)

¿Qué cuenta es el “krbtgt”?

  1. Cuenta de administrador
  2. Cuenta de servicio Kerberos
  3. Cuenta de dominio
  4. Cuenta guest

Respuesta correcta: b

Pregunta 7 (Intermedia)

¿Qué es DCSync?

  1. Sincronización de directorio
  2. Simular ser un DC para obtener hashes
  3. Un tipo de replicación
  4. Servicio de DNS

Respuesta correcta: b

Pregunta 8 (Avanzada)

Para un Golden Ticket, ¿qué hash se necesita?

  1. Hash NTLM del usuario
  2. Hash NTLM del krbtgt
  3. Hash LM del dominio
  4. Cualquier hash

Respuesta correcta: b

Pregunta 9 (Básica)

¿Qué herramienta de Impacket permite ejecutar comandos?

  1. secretsdump.py
  2. psexec.py
  3. getTGT.py
  4. samrdump.py

Respuesta correcta: b

Pregunta 10 (Intermedia)

¿Qué es un trust en Active Directory?

  1. Relación entre dominios
  2. Tipo de usuario
  3. Grupo de seguridad
  4. Politica de dominio

Respuesta correcta: a
