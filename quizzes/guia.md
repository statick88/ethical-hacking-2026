# Tests - Ethical Hacking 2026

## Tests por Unidad

Cada unidad incluye:

- 10 preguntas de opción múltiple
- 2 escenarios prácticos
- 1 ejercicio de laboratorio

## Formato de Preguntas

### Opción Múltiple

```json
{
  "pregunta": "¿Qué es un ataque de Prompt Injection?",
  "opciones": [
    "a) Inyección de código en prompts",
    "b) Manipulación de entradas para obtener respuestas no autorizadas",
    "c) Un tipo de malware",
    "d) Un protocolo de red"
  ],
  "respuesta_correcta": "b",
  "explicación": "El Prompt Injection manipula las entradas..."
}
```

### Escenario

```json
{
  "escenario": "Un pentester encuentra un formulario de login...",
  "pregunta": "¿Qué vulnerabilidad es más probable?",
  "respuesta": "SQL Injection",
  "explicación": "El formulario no valida..."
}
```

## Evaluación

### Ponderación

- Laboratorios: 40%
- Proyecto Final / Reporte: 40%
- Examen teórico: 20%

### Aprobación

- Mínimo 70% en cada test
- 80% de laboratorios completados
